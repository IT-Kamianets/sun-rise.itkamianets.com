# Guest House SunRise - Kamianets-Podilskyi

Сучасний адаптивний односторінковий сайт для гостьового будинку **SunRise**, розташованого в історичному місті Кам'янець-Подільський.

Сайт розроблений на **Angular 21** з використанням **Bulma CSS framework**.

## 🌐 Жива демо-версія
Сайт доступний за адресою: [sun-rise.itkamianets.com](https://sun-rise.itkamianets.com)

## 🛠 Технологічний стек
- **Angular 21**: Сучасна компонентна архітектура.
- **Bulma CSS**: Легкий та адаптивний фреймворк для стилізації.
- **FontAwesome**: Набір іконок для інтерфейсу.
- **GitHub Actions**: Автоматичний деплой на GitHub Pages.

## 📁 Структура проекту
- `.github/workflows/deploy.yml` - Сценарій автоматичного розгортання.
- `src/app/components/` - Компоненти сайту (Navbar, Hero, Rooms, Gallery, etc.).
- `public/assets/images/` - Папка з вашими фотографіями.

## 🚀 Як запустити локально

1. Клонуйте репозиторій:
   ```bash
   git clone https://github.com/IT-Kamianets/sun-rise.itkamianets.com.git
   ```
2. Перейдіть у папку проекту:
   ```bash
   cd sun-rise.itkamianets.com
   ```
3. Встановіть залежності:
   ```bash
   npm install
   ```
4. Запустіть сервер розробки:
   ```bash
   npm start
   ```
5. Відкрийте [http://localhost:4200](http://localhost:4200) у вашому браузері.
